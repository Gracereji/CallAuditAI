# 📞 CallAuditAI – AI Call Analysis Dashboard

CallAuditAI is an AI-powered system that analyzes customer service calls using speech-to-text and large language models.

---

## 🚀 Features

- Upload call audio
- Automatic speech-to-text transcription
- AI-based call analysis
- Performance score generation
- Interactive analytics dashboard
- AI generated call summary

---

## 🧠 How It Works

1. Upload call audio.
2. Audio is converted to text.
3. AI analyzes the transcript.
4. System generates performance metrics.

---

## 📊 Metrics Evaluated

- Customer Satisfaction
- Agent Efficiency
- Language Quality
- Communication Score
- Time Efficiency
- Bias Reduction
- Customer Emotion
- Overall Quality

---

## 🛠 Tech Stack

- Python
- Streamlit
- Groq API
- Pandas
- Matplotlib

---

## ▶️ Run the Project

Install dependencies

pip install -r requirements.txt

Run the app

streamlit run app.py
